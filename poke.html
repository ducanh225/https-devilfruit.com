Home > Paramecia > Poke Poke no Mi

Statistics

Japanese Name: ポケポケの実

Official English Name: Pocket-Pocket Fruit

Meaning: Pocket

Usage Debut: Chapter 553; Episode 461

Type: Paramecia

Current User: Blamenco

Description

The Poke Poke no Mi is a Paramecia-type Devil Fruit that allows the user to create pockets on their body. It was eaten by Blamenco.

Etymology

Poke (ポケ) comes from Poketto (ポケット), the Japanese way of pronouncing the word "pocket".

Strengths and Weaknesses

Strengths

The main strength of this fruit is that it allows the user to store various things inside pockets in their body. The objects stored inside can be much larger than the opening of the pocket, as they change size accordingly. It is stated that the user can have infinite pockets in their person, effectively giving them limitless storage capacity.

Weaknesses

No weaknesses are currently known outside the standard Devil Fruit weaknesses.

Usage

Blamenco has been shown to use this ability to store a giant hammer inside his right chin pocket.

Trivia

This fruit's power, and possibly first usage, is a reference to Hammerspace. According to Oda, Usopp would have this Devil Fruit if he was allowed one.
